1、安装uv  
  
2、打开Power shell 运行：  
uvx --from google-adk adk create --type=config my\_new\_agent  
**创建时的选项：**

-   **Model**: 选 `1` (Gemini-2.0-Flash)。
    
-   **Backend**: 选 `1` (Google AI)。
    
-   **API Key**: 粘贴你的 Key。
    

3、运行命令：uvx --from google-adk adk web  

4、打开：[http://127.0.0.1:8000](http://127.0.0.1:8000) 与本地部署的ai agent简单对话。  
  
遇到问题：我使用的是免费版的api，经常出现 429错误，资源枯竭
